Mr. H likes making dinosaur noises and teaching people about web design. Mr. H dislikes volcanos and people wearing headphones in class. Actually, he dislikes rudeness in general, not just the headphone thing. And nothing is more rude than a volcano.

  1. Create a new folder called 'madlibs' (with NO spaces) in your main folder. Create a new file inside the 'madlibs' folder called 'index.html'. Use this exact name and location.
  2. Mark up the basic HTML elements, including a title. Include a form with text boxes for adjectives and nouns.
  3. Make a second page, giving it whatever name you want, but make sure the file extension is .html . This will be the page where the 'finished' mad lib can be read.
  4. Use PHP to collect the adjectives and nouns out of the URL and insert them into the story.
  5. Style the pages attractively.
  6. Check your work using the test page:

    7. Check Your Assignment

  7. In the second half of class, consider using variables to ease the process.
